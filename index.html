Privacy Layer

Zero-Trust PII Deconstruction

Three detection layers — structural NLP, a local neural model (BERT-Small ONNX), and a statistical anomaly filter — identify and tokenize all PII on-device before any text reaches the cloud. Identity maps exist only in RAM — never written to disk, never transmitted.

Architecture

THE MODEL IS
A TOOL.

Every state transition, every question, every completion decision is made deterministically by code. The AI fills in facts. The Orchestrator makes every decision. Hallucinations are structurally impossible.
